How to populate this dashboard with live data

Run this command and paste the JSON output into the browser console, or pipe it to a file and load it: 

# Generate dashboard data from your cluster
kubectl get nodeclaims -o json | jq '{
  nodes: [.items[] | {
    name: .metadata.name,
    type: .metadata.labels["node.kubernetes.io/instance-type"] // "unknown",
    zone: .metadata.labels["topology.kubernetes.io/zone"] // "unknown",
    capacity: .metadata.labels["karpenter.sh/capacity-type"] // "unknown",
    cc_name: .metadata.annotations["karpenter.k8s.gcp/compute-class-name"] // null,
    cc_priority: .metadata.annotations["karpenter.k8s.gcp/compute-class-priority"] // null,
    created: .metadata.creationTimestamp
  }]
}'

# Or watch Karpenter logs for ComputeClass decisions
kubectl logs -n karpenter-system -l app.kubernetes.io/name=karpenter --tail=200 | grep -i "computeclass\|priority"
